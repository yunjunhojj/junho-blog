# 홍해플러스

매우 좋은 프로그램이라고 생각합니다. 그러나 비용이 매우 비싸다는 단점이 있습니다. 이러한 단점을 극복하기 위해, 이 프로그램을 무료로 제공하는 방법을 고민하고 있습니다.
